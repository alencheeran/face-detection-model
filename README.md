#  Face Detection Model

This project is a **Face Detection System** built using **OpenCV** and **Python**. It detects human faces in real-time using a webcam feed or from an image/video input. The model uses the **Haar Cascade Classifier** to identify faces accurately and efficiently.

---

## Features

- Real-time face detection using webcam  
- Detects multiple faces in an image or video  
- Option to save detected faces as cropped images  
- Lightweight and fast performance  
- Easy to integrate with other applications (e.g., attendance or emotion detection systems)

---

## Technologies Used

- Python 3.x  
- OpenCV  
- NumPy
